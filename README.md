## Featured Projects

### Finnish Job Market ETL Pipeline
Automated ETL pipeline pulling labour statistics from the Statistics Finland PxWeb API into DuckDB, with an analytics layer and tests. Handles upstream API schema changes gracefully.
<br>
`Python` `DuckDB` `SQL` `ETL` `pytest`
<br>
[View repo](https://github.com/Uththara15/finnish-job-market-pipeline)

### Course Similarity Search (BEng Thesis)
Bilingual semantic search over course descriptions using multilingual sentence embeddings (LaBSE). Built as a production service and deployed live.
<br>
`Python` `LaBSE` `sentence-transformers` `FastAPI` `Docker`
<br>
[Live demo](https://huggingface.co/uththara)

### CV Screener
LLM-powered tool that parses CVs and scores them against job requirements.
<br>
`Python` `Groq (Llama 3.1)` `Streamlit` `pypdf`
<br>
[View repo](https://github.com/Uththara15/cv-screener)

### Finland Labour Market Tracker
Interactive dashboard visualizing Finnish employment trends across multiple views.
<br>
`HTML` `CSS` `JavaScript` `Chart.js`
<br>
[View repo](https://github.com/Uththara15/finland-labour-market-tracker)

### Grounded RAG Support Bot (in progress)
Retrieval-augmented support bot with citation-backed answers and anti-hallucination guardrails, including an evaluation harness for grounding and refusal quality.
<br>
`Python` `FAISS` `sentence-transformers` `FastAPI` `Groq`
